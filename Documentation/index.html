Blazor Pos - Inventory and Sales Management System

Blazor POS is a sales, purchase and stock management software. It is fast and easy to use. You can use it for retail, grocery, shop, small business, restaurant and different types of sales management for one or multiple warehouses. There are different reports and configurations which can make it suitable for your business. Blazor POS is developed using Blazor, C#(.Net5) & MsSQL database. It’s easy to install and you’ll get free updates forever.

Blazor-Pos Installation Guide:

  1. Please Extract the Blazor-Pos.zip folder. Then you will get Blazor-Pos.zip and Documentation Folder, First read the documentation.

  2. Local Server Installation

  3. Extract the Blazor-Pos.zip file
  4. You will get Dreamer Folder
  5. Open Dreamer folder. Then you will get Dreamer.sln file
  6. Open Dreamer.sln file
  7. Clean And Rebuild Solution
  8. Change appsettings.json • ConnectionStrings tag as below:  - Server: your server name  - Database: your database name  - User id: your username  - Password: your password
  9. Now, Browse your application

  10. Plesk Server Installation.

  11. Edit LaunchsSetting
  12. Publish Project
  13. Go to plesk server
  14. Add your domain name example: (www.yourdomain.com)
  15. Add Database
  16. Import Database
  17. Upload Project
  18. Change appsettings
  19. Installation successfully done…
